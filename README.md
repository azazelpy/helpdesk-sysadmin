# Curso de Capacitación para Soporte TI y SysAdmin Junior

**Plan de estudios y recursos para formar a nuevos profesionales en Soporte de TI y Administración de Sistemas. Cubre desde hardware y redes hasta servidores y seguridad.**

---

## Descripción del Proyecto

Este repositorio contiene el material de un curso de capacitación diseñado para personas que inician su carrera en el sector de la tecnología, específicamente en roles de Soporte de TI (Help Desk) y Administración de Sistemas (SysAdmin).

El objetivo es proporcionar una base sólida y práctica que abarca los conocimientos técnicos y las habilidades blandas necesarias para un primer empleo en el área.

## Contenido del Repositorio

*   `curso_ti_sysadmin.md`: Versión del curso en formato Markdown. Es ideal para leer en GitHub, copiar, o modificar fácilmente.
*   `curso_ti_sysadmin.html`: Una versión web del mismo curso con una temática de "terminal retro" y videos de YouTube incrustados para una experiencia más interactiva. Se puede descargar y abrir en cualquier navegador.

## Temario del Curso

El curso está dividido en 7 módulos principales:

1.  **Fundamentos y el Rol del Profesional de TI:** Introducción al rol, herramientas de ticketing e ITIL.
2.  **Sporte al Usuario Final y Hardware:** Diagnóstico y resolución de problemas en equipos de escritorio y laptops.
3.  **Sistemas Operativos Cliente:** Administración de Windows, macOS y una introducción a Linux.
4.  **Redes para SysAdmin:** Fundamentos del modelo OSI, direccionamiento IP, DNS, DHCP y herramientas de diagnóstico.
5.  **Administración de Servidores y Servicios Centrales:** Introducción a la virtualización, Windows Server y Active Directory.
6.  **Seguridad Informática Práctica:** Principios de seguridad, tipos de amenazas y defensas comunes como MFA.
7.  **Introducción a la Automatización:** Primeros pasos en scripting con PowerShell y Bash.

El temario finaliza con la sugerencia de un proyecto práctico para consolidar todos los conocimientos adquiridos.
